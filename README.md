# Palate Detector (License Plate Recognition)

Desktop application for license plate recognition using AI.

## Overview

A 3-tier desktop application:
- **Frontend**: React + Vite + TypeScript
- **Backend**: Tauri v2 (Rust)
- **AI Service**: FastAPI (Python) with YOLO + OCR

## Prerequisites

- Node.js 18+ LTS
- Rust 1.70+
- Python 3.10+
- Platform-specific build tools for Tauri

## Project Structure

```
palate-detector/
├── src-tauri/     # Rust/Tauri backend
├── src/           # React frontend
└── ai-service/    # Python AI service
```
